Operations

Game

New Game

Starts a new game.


Restart

Restarts the game you are currently playing.


Numbered

Starts a new game that is based on a seed. Enter a number (up to six digits) and then you will be able to play that same game by using the same number again. The number will be displayed in the widow title.


View Removed Tiles

A window will appear that displays how many of each tile are removed from the current game board.


Exit

Exits the program.


Sound Menu

Audio On/Off

Toggles the audio either on or off.


Move

Undo

Undoes the last move made. If not previous moves have been made a message box will appear and state that there are no moves to undo.


Help

Operation

Opens this operation manual.

Game Rules

Explains the rules of MahJong.
